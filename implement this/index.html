What is a Data Structure?

A data structure is a specialized format for organizing, processing, retrieving and storing data. It defines the relationship between data elements and the operations that can be performed on them.

Campus Example: Student records organized as a linked list for dynamic insertions and deletions.
‚ö°

Why is DSA Required?

DSA provides efficient solutions to complex problems. Proper data structure selection can reduce time complexity from O(n¬≤) to O(log n), making applications faster and more scalable.

Campus Example: Hash Table for O(1) student lookup vs O(n) array search.
üìê

Static vs Dynamic DS

Static: Fixed size, allocated at compile time (Arrays).
Dynamic: Flexible size, allocated at runtime (Linked Lists, Trees).

Campus Example: Fixed course slots (Array) vs growing student list (Linked List).
üé≠

Abstract vs Concrete DS

Abstract (ADT): Defines behavior, not implementation (Stack, Queue).
Concrete: Actual implementation (Array-based Stack, Linked List Queue).

Campus Example: Priority Queue ADT implemented using Max Heap.

heapifyUp(index) {
    let parent = Math.floor((index-1)/2);
    while (index > 0 && 
        this.heap[index].priority > 
        this.heap[parent].priority) {
        this.swap(index, parent);
        index = parent;
        parent = Math.floor((index-1)/2);
    }
}

enqueue(item) {
    this.items[this.rear] = item;
    this.rear++;
    this.size++;
}

dequeue() {
    if (this.isEmpty()) return null;
    const item = this.items[this.front];
    this.front++;
    this.size--;
    return item;
}

undo() {
    if (this.undoStack.isEmpty()) 
        return null;
    const action = this.undoStack.pop();
    this.redoStack.push(action);
    return action;
}

redo() {
    if (this.redoStack.isEmpty()) 
        return null;
    const action = this.redoStack.pop();
    this.undoStack.push(action);
    return action;
}

bfs(start, end) {
    const queue = [start];
    const visited = new Set();
    const parent = {};
    
    while (queue.length > 0) {
        const node = queue.shift();
        if (node === end) 
            return this.buildPath(parent, end);
        for (const neighbor of this.adj[node]) {
            if (!visited.has(neighbor)) {
                visited.add(neighbor);
                parent[neighbor] = node;
                queue.push(neighbor);
            }
        }
    }
}

insert(time, resource) {
    const newNode = new TreeNode(time, resource);
    if (!this.root) {
        this.root = newNode;
        return true;
    }
    return this._insertNode(this.root, newNode);
}

inorder(node, result = []) {
    if (node) {
        this.inorder(node.left, result);
        result.push(node);
        this.inorder(node.right, result);
    }
    return result;
}
